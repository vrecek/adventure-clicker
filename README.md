# Adventure clicker game
Basic clicker game with everything what normal clicker should have

# Screen support
Game was tested on 1080p laptop changed to 1280x720 resolution, so on smaller screens, especially mobile you may encounter things that shouldnt be or you wont be able to see actual thing at all, only separate html page with above information

# TypeScript, Scss, HTML
Project made mainly to practice TS

# Features
Animated statistics menu,
Based on rng active item which gives money if succeed, otherwise lose some of cash,
Obviously upgrades that give some boosts
